Changes since release 9.22

  • Add Python 3.8 support.

  • model.read() now understands mmCIF format author-provided chain IDs, which are used instead of the canonical chain ID if available.

  • model.read() now allows for chain IDs longer than 1 in model_segment when reading mmCIF files.
